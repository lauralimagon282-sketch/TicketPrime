Este é o backend do sistema TicketPrime, desenvolvido como uma Minimal API em .NET utilizando Dapper para persistência de dados.

## Como Executar o Projeto

Para compilar e rodar a API, certifique-se de ter o SDK do .NET instalado, navegue até a pasta do código-fonte e execute:

```bash
cd src
dotnet run
```
